CEM
Component Element Modifier

Components

Prefixs

Every components class need to be prefixed! In this document, the prefix used will be pre.

Wrong 

.dropdown

Correct 

.pre-dropdown

Elements

Every elements in a component need to contain the name of their parent component. 
<div class="pre-dropdown">
<button class="pre-dropdown__trigger-btn"></button>
</div>

Correct 

.pre-dropdown__trigger-btn
to select the component in the less file 
.pre-dropdown  {
	&__trigger-btn {
		//
	}
}

Modifiers

Context Modifier

If you want to add a context modifier add two dashs at the end of your element class. 

.pre-dropdown--register-form
,pre-dropdown--welcome-page

State Modifier

State modifier start with: 

.is-
.has-
.is-open
.is-checked
.has-menu
.has-children

Javascript

.js- classes are use only for Javascript selector. Never attach CSS rules to a class starting with js- 
js-pre-dropdown__element__subelement
